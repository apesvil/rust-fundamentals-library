This repository has a very simple Rust library created from the Coursera course Rust Fundamentals.

It was initially created using the command

```
cargo new my_library --lib
```

To build the library type

```
cargo build
```

The compiled library file will appear in the `target/debug` directory.

## Documentation

To build the documentation we used the command

```
cargo doc
```

The documentation of the library can be found in the `target/doc` directory.

To open it in the browser, run

```
cargo doc --open
```
